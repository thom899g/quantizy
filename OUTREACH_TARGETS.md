# Quantizy Outreach Targets

Use this as a ranked outreach batch. Send one relevant, human message per target
or public submission channel. Do not automate repeated DMs.

## Priority 1: Best Fit

| Target | Why it fits | Angle |
| --- | --- | --- |
| Contra Collective | Recent Apple Silicon MLX vs llama.cpp coverage | Offer Quantizy as a fit-check layer before users pick runtime/model size. |
| SitePoint AI | Publishes local LLM and Apple Silicon guides | Pitch a practical paid beta for readers deciding what their Mac can run. |
| Towards AI | Covers MLX/Ollama performance and local AI tradeoffs | Ask for an honest review of memory guardrails and current-RAM fit checks. |
| YouTube: MLX vs GGUF / Apple Silicon benchmark creators | Already tests Mac local-AI runtimes | Offer a reviewer license and ask them to test whether Quantizy prevents bad downloads. |
| LocalLLaMA Reddit weekly threads | Audience already runs local models and cares about RAM | Post only where self-promotion is allowed; lead with benchmark/fit-check evidence. |
| Ollama Reddit / LM Studio communities | Users struggle with model choice and memory limits | Ask for testers with 16 GB, 32 GB, and 64 GB Macs. |
